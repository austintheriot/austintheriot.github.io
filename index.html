3D Particle Application

I created this project using the visual programming environment Max 8. This 3D Particle Attractor Application provides an interactive tool for viewing 3D "strange attractor" particle simulations. The speeds of various strange attractors can be adjusted and combined to form interesting particle effects.

This project later became the basis for one of my multimedia compositions for percussion and computer with live, projected visuals, called Light Flower.

Built Using:

  1. Max 8
  2. OpenGL 3
  3. OpenGL Shading Language (GLSL)
  4. A custom-coded shader

Computer-Assisted Composition

Computer-Assisted Composition involves the use of a computer in the creation of musical ideas, especially for classical composition. I built this Canon Creation Tool using Max 8 to assist with the creation of so-called musical "canons" or "rounds." These compositions, in which one voice follows the other in exact imitation, are notoriously difficult to compose and plan. This application eases that process for composers by altering every voice in real time as the original voice is changed and by providing baked-in musical transformations that can be easily selected. The final result can the be played back immediately or exported into other music notation softwares such as Sibelius or Finale.
